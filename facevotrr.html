Project description

"An Online voting system using Face Recognition"
A web-based voting application based on Fisherface, Eigenface, and Surf algorithms developed using Python and Flask. The website used usual front-end tools such as Html, CSS, and JS, and for the back-end, it used C++ libraries such as Dlib, and Python libraries such as OpenCV and Flask Framework to connect the full stack. A local database is used to handle the data in smaller size.
It consists of three Alorithms:
Eigenface algorithm is capable of recognizing and detecting faces based on their variance in a set of images. Among multiple images captured by the system, Eigenface selects the mean image.
Fisherface algorithm identifies principal components that differentiate individuals and also immune to noise-induced images and the blurring effect on the image.
Surf algorithm perform recognition in three sections:
1. Detecting the interest points by constructing the hessian matrix,
2. Describing the interest points by extracting the mathematical values from the constructed hessian matrix,
3. Matching the intrest points

Tech Stack

ToolKit: Anaconda, Jupyter
Frontend: HTML, CSS, Javascrpit
Backend: Python
Database: Csv format
