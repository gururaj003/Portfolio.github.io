Project description:
SFDC project based on Salesforce Service - FSL (Field Service Lightning).
It involves the maintenance of elevators, escalators, etc., as well as the repair of any emergencies caused by these assets.
The activities I was involved in:
● Implemented a custom notification system within Salesforce to prompt users on instant record updates.
● Developed a custom functionality to send SMS via platform events using the salesforce ootb messaging feature.
● Crafted and deployed a complex business process of Timesheet and expense functionality using apex triggers, queueable class and salesforce mobile flow. Automated the approval process, adjusting the time sheet dates on holidays by invoking the apex (batch class) with schedule trigger flow.
● Assisted in development of salesforce mobile solutions. Developed complex custom mobile (FSL Screen) flows. Optimized custom components during the UAT and SIT phase before production release to avoid code errors.
● Contributed to the configuration of the FSL feature in the Sandbox environment
